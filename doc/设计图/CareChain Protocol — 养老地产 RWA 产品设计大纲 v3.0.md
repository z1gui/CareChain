## 一、项目定位与核心命题

**CareChain** 是建立在 Solana 上的养老地产 RWA 协议，将停工/低效商业物业转型为可投资的养老设施，通过 NFT 将床位使用权与收益权代币化。协议引入**优先队列机制**：持有 BedRight NFT 的用户可直接入住，普通申请者需排队等待，持有 $CARE 代币可跳过队列——让代币拥有真实的刚需消耗场景。

**一句话 Pitch**：把养老床位变成可投资、可交易、可使用的链上资产——NFT 持有者直接入住，普通家庭排队等待，不想排队就买 $CARE。

### 核心创新点

| 维度 | 传统养老投资 | CareChain |
|------|-------------|-----------|
| 最低投资门槛 | 数百万人民币（机构） | ~$12,000 USDC（一张床位 NFT） |
| 流动性 | 极低（房产锁定） | 高（NFT 二级市场可转让） |
| 收益透明度 | 黑箱（财报滞后） | 链上实时分润，公开可查 |
| 使用权 | 无 | 持有 NFT 直接入住，无需排队 |
| 普通家庭入住 | 漫长等待 | 燃烧 $CARE 代币可提升优先级 |
| 退出机制 | 复杂（资产交割） | NFT 市场直接卖出 |

## 二、市场机会与底层资产

### 两个万亿级市场的交叉点

**养老产业缺口**：中国 2035 年 60 岁以上人口将突破 4 亿，养老床位缺口保守估计超过 800 万张，现有供给严重不足，优质养老设施一床难求，排队等待已是常态。

**商业地产困境**：2022 年以来中国商业地产开工率持续下降，大量物业停工或空置，业主急需融资和转型路径。佛山周边珠三角地区此类资产尤为集中。

**RWA 赛道爆发**：链上 RWA 资产规模在 2026 年突破 300 亿美元，房地产代币化是增速最快的子赛道，但针对养老地产的协议几乎空白。

### 首发底层资产：佛山澳门城

CareChain 的首个锚定资产为真实的停工商业街区改造项目：
- **资产现状**：位于佛山三龙湾南海片区，原规划为澳门特色商业街区，已投入 1 亿元人民币沉没成本，目前处于停工状态。
- **改造规划**：地上建筑面积 50,000 ㎡（有效使用面积约 35,000 ㎡），计划融资 1 亿元人民币（约 1,330 万 USDC）盘活资产。
- **床位规划**：根据国家标准（≥35㎡/床），规划 **800 张**高端养老床位（平均 43.8㎡/床），兼顾高端品质与经济效率。

## 三、双轨入住机制（核心设计）

这是 CareChain 区别于所有现有 RWA 协议的关键创新——**将「床位稀缺性」转化为代币的真实需求驱动力**。

### 3.1 三通道优先队列

```
床位空出时，按以下优先级顺序分配：

P1 ─ VIP 通道（立即入住）
     └── BedRight NFT 持有者（使用权已锁定）

P2 ─ 快速通道（24–72 小时）
     └── 燃烧 $CARE 代币的申请者
         燃烧量越多 → 队列优先级越高

P3 ─ 标准队列（先到先得，等待时间不定）
     └── 未持有任何代币的普通申请者
```

### 3.2 $CARE 动态燃烧定价

跳队成本根据当前 P3 队列长度动态调整，形成市场化定价：

| P3 队列人数 | 燃烧倍率 | 每天跳队成本 | 信号含义 |
|------------|---------|------------|---------|
| < 20 人 | 1× | 15 $CARE | 床位充裕 |
| 20–80 人 | 1.5× | 22.5 $CARE | 供需偏紧 |
| > 80 人 | 2× | 30 $CARE | 稀缺溢价 |

动态定价的双重意义：队列越长燃烧越多形成正反馈；设施受欢迎程度直接反映在代币消耗量上，为投资者提供真实的设施价值信号。

### 3.3 代币经济飞轮

```
更多家庭申请养老床位
        ↓
  P3 标准队列变长
        ↓
  $CARE 燃烧需求上升
        ↓
   $CARE 流通量减少
        ↓
  投资者收益上涨 + 代币升值
        ↓
  更多资金流入协议
        ↓
  更多设施加入 CareChain
        ↓
  更多床位供应 → 吸引更多家庭 ←（循环）
```

这是一个**真实需求驱动**的飞轮，底层是「老人需要养老床位」这一不可替代的社会刚需，而非炒作。

### 4.2 四大核心合约（Solana / Anchor）

#### A. BedRight NFT Program
每枚 NFT 对应一张具体床位，基于 **Metaplex Core** 标准铸造：

```json
{
  "bed_id": "FSH-A301",
  "facility": "佛山澳门城康养中心",
  "room_type": "自理型标准间",
  "care_tier": "生活自理",
  "annual_yield_rate": "16.4%",
  "face_value_usdc": 12000,
  "status": "yield_mode"
}
```

持有者可在「**收益模式**」与「**入住模式**」间切换，切换需 30 天冷却期，防止短期套利。

#### B. PriorityQueue Program
- 维护三个优先级通道的链上等待名单
- 接收 $CARE 燃烧指令，实时更新申请者队列排名
- 床位空出事件触发时，自动按优先级顺序发出入住邀请
- 配合 Pyth Oracle 接入设施实时占用率，动态调整定价倍率

#### C. YieldVault Program
- 设施运营方每月将 USDC 注入合约资金池
- 按所有流通 BedRight NFT 持有者比例自动分配
- $CARE 燃烧收入的 50% 也注入 YieldVault，**增厚 NFT 持有者的额外收益**

#### D. $CARE Token Program
- SPL 标准代币，总量固定上限（通缩设计）
- 燃烧销毁机制：每笔跳队消耗的 $CARE **永久销毁**，不返还协议
- 质押功能：NFT 持有者质押 $CARE 可获得额外收益加成

## 五、代币经济设计

### 5.1 BedRight NFT 系列（佛山澳门城，800 床）

目标融资 1 亿元人民币（约 1,330 万 USDC），定价参考广深佛高端养老市场标准。

| 类型 | 数量 | 面值 (USDC) | 预期 APY (稳态) | 特权 |
|------|------|-------------|-----------------|------|
| 自理型标准间 | 400 枚 | $12,000 | 16.4% | 基础收益 + P1 直接入住 |
| 半护理型套间 | 250 枚 | $22,000 | 16.4% | 优先选房 + 治理加权票 |
| 全护理型记忆照护 | 100 枚 | $35,000 | 16.4% | 专业照护 + 治理双倍票 |
| VIP 豪华套房 | 50 枚 | $50,000 | 16.4% | 顶级定制 + 顶级治理权 |

> *注：16.4% 的高 APY 得益于引入了康复理疗、认知症专护等高利润的**增值服务**（占总收入 15.9%）。*

### 5.2 $CARE 代币分配

| 用途 | 比例 | 说明 |
|------|------|------|
| 社区空投（早期用户） | 25% | 激励早期家庭用户和 NFT 投资者 |
| 流动性挖矿 | 20% | NFT 持有者质押获得 $CARE 奖励 |
| 生态激励 | 20% | 设施运营方激励、数据上报奖励 |
| 核心团队 | 15% | 2 年线性解锁 |
| 协议国库 | 10% | DAO 管理，用于紧急补偿和生态建设 |
| 战略投资者 | 10% | 早期支持机构 |

**通缩机制**：$CARE 总量固定，每次跳队燃烧永久销毁。设施占用率越高 → 队列越长 → 燃烧越多 → 流通量越少 → 稀缺性越强。

### 5.3 月度收益分配

```
设施月度收入 = 床位费 + 护理费 + 增值服务费
                        ↓
   ┌────────────────────────────────────┐
   │  60% → YieldVault → NFT 持有者分润  │  ← 主要收益
   │  +另加 $CARE 燃烧收入的 50%         │  ← 额外增厚
   │  15% → 设施运营再投入               │
   │  10% → 协议国库（DAO 基金）          │
   │   5% → 保险储备金（空置补偿机制）    │
   │   5% → $CARE 燃烧奖励               │
   │   5% → 协议手续费                   │
   └────────────────────────────────────┘
```

## 六、用户体验流程

### 路径 A：投资者（Web3 用户）

```
① 连接 Phantom 钱包
② 浏览设施 Dashboard（占用率、分润记录、床位地图、P3 队列长度）
③ 选择床位类型 → USDC 购买 BedRight NFT
④ NFT 到账，进入「收益模式」自动计息
⑤ 每月 1 日收取 USDC 分润（含 $CARE 燃烧增益）
⑥ 质押 $CARE 获得额外收益加成（可选）
```

### 路径 B：有规划的家庭（Web3 用户）

```
① 选择「养老规划模式」
② 输入父母预计入住年份 → 推荐床位类型
③ 購入 BedRight NFT（持有期间持续获得收益）
④ 入住时切换「入住模式」→ 直接进入 P1 通道
⑤ 入住确认后 NFT 转为 SBT（灵魂绑定），记录护理履历
```

### 路径 C：普通家庭（非 Web3 用户）

```
① 在设施官网提交入住申请（无需钱包）
② 进入 P3 标准队列，收到预计等待时间通知
③ 若想提前入住 → 引导至 $CARE 购买页面（法币通道）
④ 选择跳队天数 → 系统自动代为燃烧对应 $CARE
⑤ 升入 P2 快速通道 → 24-72 小时内确认入住
```

> **设计亮点**：路径 C 的普通家庭完全不需要理解 Web3，他们只需要知道「交费可以提前入住」——$CARE 的链上燃烧对他们是透明无感的。

| 数据索引 | Helius RPC | 高性能 NFT/资产索引，驱动实时 Dashboard |

## 八、商业模式

### 协议收入来源

| 收入类型 | 费率 | 触发条件 |
|----------|------|----------|
| NFT 铸造手续费 | 面值的 1.5% | 每次新床位 NFT 铸造 |
| 协议管理费 | NOI 的 5% | 每次 YieldVault 结算 |
| 二级市场版税 | 成交价的 2.5% | NFT 每次转让 |
| 跳队手续费 | 燃烧 $CARE 金额的 5% | 每笔跳队操作（法币通道收取） |

### 规模化路径

**Phase 1（黑客松 MVP）**：佛山澳门城改造项目，800 床位，验证铸造、分润、三通道队列全流程。

**Phase 2（6 个月后）**：珠三角 3-5 个合作设施，累计 3,000+ 床位，上线 $CARE 代币与跳队机制。

**Phase 3（12 个月后）**：扩展至东南亚（泰国、马来西亚有大量华人养老需求），成为「Solana 上的养老地产 REITs + 入住权协议」。

## 十、黑客松 Pitch 要点

### 为什么 CareChain 能赢？

1. **真实问题，不是造概念**：佛山澳门城 1 亿沉没成本的停工物业，与中国 800 万养老床位缺口形成强烈对比，这是真实存在的社会问题。
2. **代币有刚需**：$CARE 的价值来源于「不想排队的家庭」——这是任何市场周期都存在的需求，不依赖炒作。
3. **RWA 细分空白**：现有 RWA 协议全部聚焦美国/欧洲，中国+东南亚养老地产无竞争者，市场蓝海清晰。
4. **NFT 创新用途**：大多数 NFT 是纯金融工具，BedRight NFT 可「消费」（兑换实际服务），开创 NFT 即服务凭证的新范式。
5. **Web2 用户可用**：路径 C 的普通家庭无需钱包，用法币即可购买跳队资格，真正实现 Web3 Mass Adoption。

### 核心 Pitch 金句

> *"你父母需要一张养老床位。传统方式：排队等 18 个月。CareChain 方式：持有 NFT，直接入住；没有 NFT？排队。不想排队，买 $CARE。这不是 DeFi 炒作——这是用 Web3 盘活 1 亿停工资产，解决 4 亿老人的真实问题。"*
